AI Mapping

建議搭配哪一個 AI？

建議優先搭配 Gemini 使用。原學習流程的 AI 篇偏向 Gemini 生態，例如 Gems、Gemini Live、Canvas、Guided Learning、Quiz / Flashcards。

但這個工具不是只能用 Gemini。ChatGPT、Claude 或其他 AI 也可以用，只要把專有功能換成相近用法。

Gem / Gems
自訂 AI 英文教練、Custom GPT、Project 或長期對話設定。
Gemini Live
語音對話、Voice Mode、一般口說角色扮演。
Canvas
寫作編輯區、Canvas、Artifacts 或一般對話改稿。
Guided Learning
要求 AI 逐步教你，不要直接給答案。
資料載入中。如果你是用 file:// 直接打開，瀏覽器可能阻擋 JSON 載入；請上傳到 GitHub Pages 後使用。
Today Minimum Mode

今天不知道練什麼？照這 15 分鐘做

  1. 用 C01 告訴 AI 你的程度與目標。
  2. 沒有材料就用 C08 選「請 AI 幫我生成」。
  3. 想練聽力用 C18；想練閱讀用 C25；想練寫作用 C41。
  4. 最後用 C58 留下今天錯誤，明天再練一次。
Resource Navigator

資源導航：先選目標，再選材料

資源不要收藏，請直接使用。每個資源都要知道：適合誰、怎麼用、搭配哪張 Prompt 卡。

Prompt Toolbox

Prompt 卡：打開、複製、貼到 AI

本區共 60 張卡。每張卡都包含主 Prompt、二次追問，以及 AI 回覆後要做的事。

手機若無法自動複製，請長按 Prompt 文字框，全選後手動複製。

30-Day Plan

30 天練習節奏

每天只需要完成一個可驗收輸出。不要把計畫做成壓力表。
