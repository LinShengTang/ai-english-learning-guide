## 判斷

v2.1 比照常見開源 GitHub Pages 做法：

1. 首頁先顯示工具用途與開始入口。
2. 首頁下方立即放一段簡短 Attribution，不收合。
3. About / README / LICENSE-NOTE / appendix/attribution.html 放完整來源、授權與非官方改編聲明。
4. 不把完整授權聲明放在第一屏最頂端，避免首頁像法律頁。
5. Footer / About / README 保留完整 attribution。
