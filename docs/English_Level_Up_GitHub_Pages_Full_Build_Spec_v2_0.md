# English Level Up GitHub Pages Full Build Spec v2.0

## 目標

將 v1.0 單檔互動工具升級成 GitHub Pages 多檔版，提升完整性、可維護性與公開分享品質。

## 架構

- index.html：唯一分享入口
- assets/style.css：樣式
- assets/app.js：資料載入、渲染、篩選、複製
- data/chapters.json：七大主章
- data/resources.json：Resource Navigator
- data/prompt-cards.json：60 張 Prompt 卡
- data/plan.json：30 天計畫
- appendix/source-map.html：來源對照
- appendix/attribution.html：授權與署名

## 侵權風險 Gate

PASS 條件：

1. About 顯示原項目與 CC BY-NC 4.0
2. README 與 LICENSE-NOTE 保留非官方 / 非商業聲明
3. 不重發第三方圖片、影片、商標或課程內容
4. 不宣稱原作者背書
5. Word Lists 不作為一般前台頁
6. 不作為商業用途
