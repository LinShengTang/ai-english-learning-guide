# AI 英語學習互動指南

GitHub Pages 前台：  
https://linshengtang.github.io/ai-english-learning-guide/

這是一個非商業英文學習輔助工具，將 byoungd 的開源項目 English-level-up-tips 整理為繁體中文互動學習流程、單字素材庫、學習材料選單與可複製 Prompt 卡。

## 使用方式

第一次使用，建議先打開前台，從「今日任務」開始，用 15 分鐘完成一次最小練習。  
有材料時，可以從「學習材料」選一張資源卡；沒有材料時，可以從「單字素材庫」選一組詞開始。  
要找 Prompt 時，請進入「Prompt 卡」區塊，使用章節索引快速跳到 C01–C60。
